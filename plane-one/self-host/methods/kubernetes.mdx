## Quick Setup
```
# It will setup local Minio, Postgress, Redis with longhorn as storage class driver
# It will skip installing SSL certificates
# Domain Name once attached to license cannot be changedPLANE_HELM_CHART=$(ls -t *.tgz | head -n1)

helm install one-app ${PLANE_HELM_CHART} \
--create-namespace \
--namespace plane-one \
--set dockerRegistry.loginid=${REG_USER_ID} \
--set dockerRegistry.password=${REG_PASSWORD} \
--set license.licenseKey=${LICENSE_KEY} \
--set license.licenseDomain=${DOMAIN_NAME} \
--set license.licenseServer=https://crew42.prime.plane.tools \
--set planeVersion=${PLANE_VERSION} \
--set ingress.ingressClass=nginx \
--set env.storageClass=longhorn \
--timeout 10m \
--wait \
--wait-for-jobs
```

## Advanced Setup
```
helm show values ${PLANE_HELM_CHART} > values.yaml
vi values.yaml

PLANE_HELM_CHART=$(ls -t *.tgz | head -n1)

helm install one-app ${PLANE_HELM_CHART} \
--create-namespace \
--namespace plane-one \
-f values.yaml \
--timeout 10m \
--wait \
--wait-for-jobs
```
